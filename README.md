**Tech Stack**

   Microsoft Azure AI Foundry
   ASP.NET Web API
   .NET (C#)
   Visual Studio
   RESTful APIs

**Project Overview**

This project demonstrates the creation of an interactive chatbot using Microsoft Azure AI Foundry and .NET.
The chatbot is implemented as an ASP.NET Web API that exposes API endpoints to provide intelligent responses using Microsoft AI services.

The application is developed in C# using Visual Studio and is designed to be scalable, secure, and easy to integrate with other web or client applications.
